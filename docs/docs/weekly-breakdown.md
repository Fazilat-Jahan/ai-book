# Weekly Breakdown

This 12-week program is structured to provide a comprehensive and progressive learning experience. Each module spans three weeks, building upon the knowledge and skills from the previous one.

## Module 1: Foundations of AI and Robotics (Weeks 1-3)

-   **Week 1:** Introduction to AI and Robotics, Mathematical Foundations.
-   **Week 2:** Python for AI, and essential libraries.
-   **Week 3:** Module 1 Project: Building a simple AI application.

## Module 2: Computer Vision for Robotics (Weeks 4-6)

-   **Week 4:** Introduction to Computer Vision, Image Features and Filters.
-   **Week 5:** Object Detection and Recognition with Deep Learning.
-   **Week 6:** Module 2 Project: Implementing a real-time object detection system.

## Module 3: NLP and Reinforcement Learning (Weeks 7-9)

-   **Week 7:** Introduction to NLP and Text Processing.
-   **Week 8:** Introduction to Reinforcement Learning and Q-Learning.
-   **Week 9:** Module 3 Project: Training an RL agent to perform a simple task.

## Module 4: Humanoid Robotics and Real-World Control (Weeks 10-12)

-   **Week 10:** Introduction to Humanoid Robotics, Kinematics and Dynamics.
-   **Week 11:** Bipedal Locomotion and Manipulation.
-   **Week 12:** Final Project: Designing and simulating a complete AI-driven humanoid robot application.
